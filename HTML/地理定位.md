# 地理定位面试笔记

## 1. HTML5 地理定位是什么

### 问题

HTML5 地理定位是什么？

### 回答

HTML5 提供了 Geolocation API，可以让网页在用户授权后获取当前地理位置信息。

---

## 2. 常用 API 有哪些

### 问题

地理定位常用 API 有哪些？

### 回答

- `getCurrentPosition`
- `watchPosition`
- `clearWatch`

---

## 3. 使用地理定位要注意什么

### 问题

使用 Geolocation API 要注意什么？

### 回答

- 需要用户授权
- 一般要求安全上下文，也就是 HTTPS
- 定位精度和设备、网络环境有关

## 4. 速记版

- 地理定位需要授权
- 常用 `getCurrentPosition`
- 通常要求 HTTPS

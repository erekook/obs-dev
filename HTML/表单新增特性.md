# 表单新增特性面试笔记

## 1. HTML5 表单有哪些增强

### 问题

HTML5 对表单做了哪些增强？

### 回答

主要有两类：

- 新增 input 类型
- 新增表单属性和原生校验能力

---

## 2. 新增 input 类型有哪些

### 问题

HTML5 新增了哪些常见 input 类型？

### 回答

常见有：

- `email`
- `url`
- `number`
- `tel`
- `search`
- `range`
- `color`
- `date`
- `time`
- `month`
- `week`

---

## 3. 新增属性有哪些

### 问题

HTML5 表单新增了哪些常见属性？

### 回答

常见有：

- `placeholder`
- `required`
- `autofocus`
- `autocomplete`
- `pattern`
- `multiple`
- `min`
- `max`
- `step`

---

## 4. HTML5 表单增强的好处

### 问题

HTML5 表单增强有什么实际好处？

### 回答

- 减少手写校验代码
- 提升用户输入体验
- 让移动端弹出更合适的键盘
- 提高表单语义和可访问性

## 5. 速记版

- 新增类型：`email` `url` `number` `date` `range`
- 新增属性：`placeholder` `required` `autofocus` `pattern`
- 优点是原生校验更强，体验更好

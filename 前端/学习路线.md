# 一、前端学习路线（整理版）

## 1. 基础阶段

- HTML / HTML5
    
- CSS / CSS3
    
- JavaScript（ES6/ES7/ES8）
    

👉 目标：掌握页面结构、样式、基础交互

---

## 2. 进阶工具链

- CSS预处理器：Less / Sass
    
- JS编译工具：Babel
    
- 类型系统：TypeScript
    
- 构建工具：Webpack / Vite
    

👉 目标：理解现代前端工程化

---

## 3. 框架体系（核心）

- Vue 或 React（二选一或都学）
    
- 全家桶：
    
    - 路由（Vue Router / React Router）
        
    - 状态管理（Vuex / Pinia / Redux）
        
    - UI框架（Element / Ant Design 等）
        

👉 本质：  
从「框架 → 路由 → 数据管理 → UI组件」的一整套体系

---

## 4. CLI & 工程能力

- Vue CLI / React CLI（或 Vite 脚手架）
    

👉 目标：掌握项目初始化与构建流程

---

## 5. 原理深入（非常关键）

- 框架原理（响应式、虚拟DOM等）
    
- 工程原理（打包、模块化）
    

⚠️ 重点说明：

> 不是为了面试，而是提升代码能力和技术视野  
> 很多高级写法只能在优质源码中学到

---

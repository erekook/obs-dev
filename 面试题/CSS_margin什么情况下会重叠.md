# 题目
margin 什么情况下会重叠？

# 你的回答
先主动追问了这个问题。

# 面试点评
这是 BFC 相关高频题，重点要知道是“垂直方向块级元素”的 margin 折叠，而不是所有 margin 都会重叠。

# 回答要点
- margin 重叠主要发生在垂直方向
- 常见于相邻兄弟块级元素
- 父子块级元素也可能发生重叠
- 空块级元素的上下 margin 也可能重叠
- 水平方向 margin 一般不会重叠

# 参考答案
margin 重叠通常发生在垂直方向上的块级元素之间，比如相邻兄弟元素、父子块级元素、空块级元素。
本质上是相邻的垂直外边距合并成一个外边距，结果通常取较大的那个值，而不是简单相加。水平方向的 margin 一般不会重叠。

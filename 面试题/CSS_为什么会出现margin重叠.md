# 题目
为什么会出现 margin 重叠？

# 你的回答
先主动追问了这个问题。

# 面试点评
理解这题的关键在于知道 margin 是元素外部间距，浏览器在垂直排版块级元素时会把相邻外边距进行折叠，而不是简单相加。

# 回答要点
- margin 是元素外部间距
- 垂直方向相邻块级元素的外边距可能折叠
- 浏览器会把相邻外边距合并，而不是简单累加
- 这样设计是为了让块级元素垂直排版更自然
- 父子元素没有边界隔开时也可能发生重叠

# 参考答案
因为 margin 表示元素外部间距，在垂直方向上相邻的块级元素之间，浏览器会把相邻外边距进行合并，而不是简单相加，这就是 margin 重叠。
这样设计是为了让块级元素的垂直排版更自然，避免上下间距叠加过大。
父子元素如果没有 padding、border 或 BFC 隔开，也可能发生这种重叠。

# 题目
`Object.create()` 做了什么？

# 你的回答
创建一个新对象，并把传入的对象作为新对象的原型。

# 面试点评
这版已经可以了。再补一句“新对象可以通过原型链访问传入对象上的属性和方法”会更完整。

# 回答要点
- 创建一个新对象
- 把传入对象作为新对象的原型
- 新对象可以沿原型链访问该原型上的属性和方法

# 参考答案
`Object.create()` 用于创建一个新对象，并把传入的对象作为这个新对象的原型。也就是说，新对象会通过原型链继承传入对象上的属性和方法。

# 题目
`new` 一个对象时发生了什么？

# 你的回答
new 在执行时，首先会创建一个新的空对象，把这个新对象的隐式原型指向构造函数的 prototype，接着执行构造函数，并把构造函数内部的 this 绑定到这个新对象上，最后返回这个新对象。如果构造函数显式返回一个引用类型，则返回该引用类型，否则返回这个新对象。

# 面试点评
这版已经达到面试可用水平。再补一句“会把 new 后的参数传给构造函数”是加分项。

# 回答要点
- 创建一个新的空对象
- 将对象的隐式原型指向构造函数的 prototype
- 执行构造函数并绑定 this
- 传入 new 后面的参数
- 返回新对象或构造函数显式返回的引用类型

# 参考答案
`new` 在执行时，首先会创建一个新的空对象；然后把这个新对象的隐式原型指向构造函数的 `prototype`；接着执行构造函数，并把构造函数内部的 `this` 绑定到这个新对象上，同时传入 `new` 后面的参数；最后返回这个新对象。如果构造函数显式返回一个引用类型，则返回该引用类型，否则返回这个新对象。

# 题目
什么是原型链？

# 你的回答
每个对象都可以通过原型对象关联到另一个对象，当访问属性时，如果自身没有找到，会沿着原型往上找，这就是原型链。

# 面试点评
方向正确，已经说出了原型链的核心。还可以补充“原型链最终会查找到 null 为止”，以及“为什么叫链”，也就是原型对象本身也可能有自己的原型，层层相连。

# 回答要点
- 对象可以通过原型关联到另一个对象
- 访问属性时，先找自身
- 自身没有会沿原型往上查找
- 原型对象本身也可能有自己的原型
- 最终会找到 null 为止

# 参考答案
原型链是 JavaScript 中对象之间基于原型进行关联形成的一条链式查找结构。每个对象都有自己的原型对象，当访问一个对象的属性时，如果对象本身没有这个属性，就会沿着它的原型对象继续往上查找。而原型对象本身也可能有自己的原型，这样一层一层向上连接起来，就形成了原型链。如果一直找到顶层仍然没有，一般会到 `null` 为止，然后返回 `undefined`。

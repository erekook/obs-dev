- 重绘 Repaint
重绘指的是元素改变了样式，在不影响其布局的情况下，浏览器重新绘制元素的过程。例如修改了元素的背景颜色，字体颜色等
- 回流 Reflow
回流是指元素的布局属性发生了改变，需要重新计算元素在页面中的布局位置时，浏览器重新布局的过程。例如修改元素的宽高，位置等
- 回流的成本比重绘高很多，因为需要重新计算元素的几何属性和页面布局，而重绘只要重新绘制已经计算好的元素样式
- 如果减少回流？
	- 使用CSS动画代替Javascript动画
	- 使用translate3d开启硬件加速
	- 使用requestAnimationFrame
	- 使用文档片段
	- 让元素脱离文档流，position：absolute；position:fixed;float:left
	- visibility:hidden代替display:none
	- 避免频繁操作影响元素布局的样式属性
